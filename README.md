A Ideia (O Problema e a Solução)
O Problema: Desperdício de comida e falta de criatividade para cozinhar com o que sobrou na geladeira.

A Solução: Uma IA que olha para os ingredientes disponíveis e inventa receitas na hora.

Como Funciona (As 3 Etapas)
Entrada (Input): O usuário tira uma foto do interior da geladeira.

Processamento (IA): O modelo multimodal (visão + texto) identifica os alimentos e cruza os dados para gerar receitas.

Saída (Output): A tela mostra quais ingredientes foram encontrados e dá de 2 a 3 opções de pratos com o passo a passo.

Tecnologias Utilizadas
Linguagem: Python

Cérebro da IA: API do Google Gemini (para "enxergar" a foto e escrever as receitas)
